    Contenido principal del body

    la etiqueta html5 es una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la página. Unicamente puede haber una sola etiqueta como esta en todo el documento el contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de pagina, los menues de navegacions y lo pies de pagina). Es usado por buscadores pa indexar conviene usar aqui texto que ayuden a encontrar la pagina luego de procesos de indexacion de los buscadores.

    Sección etiquetas semanticas estructurales mas importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICULE
  • NAV
  • ADDRESS

    • las etiquetas semanticas estructura esta relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dntro del documento

    reglas para etiquetas semanticas

  • los distintos tipos de contenidos se pueden asociar con las etiqUetas SECTION
  • las unidades de contenidos se asocian con las etiquesas ARTICLE. O sea un SECTION unida a multiples ARTICLES
  • Las etiques HEADER y FOOTER pueden ser anidadas destrpo de otras HEADER o FOOTER. O sea un FOOTER podria anidar otre HEADER o FOOTER
  • La eqiuqeta NAV debe ser única y corresponde a los elementos de navegacion del documento
  • Reservar etiquetas H1 para usar dentro de etiquetas top level como MAIN, HEADER o FOOTER
